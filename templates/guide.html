Guide


Isolation Forest is an outlier detection technique that identifies anomalies instead of normal observations. Similarly to Random Forest, it is built on an ensemble of binary (isolation) trees. It can be scaled up to handle large, high-dimensional datasets. An explanation of the isolation forest algorithm parameters in the sklearn library can be seen in the following link :
